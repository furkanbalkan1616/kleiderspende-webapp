# Kleiderspenden Registrierung

Diese Webanwendung wurde im Rahmen des Moduls
„Programmierung von Webanwendungsoberflächen (IPWA01-01)“ erstellt.

## Funktionen

- Registrierung von Kleiderspenden
- Auswahl eines Krisengebiets
- Übergabe an Geschäftsstelle oder Abholung
- Postleitzahl-Validierung
- Bestätigungsseite mit Datum und Uhrzeit

## Technologien

- HTML5
- CSS / Bootstrap
- JavaScript
- Git & GitHub
